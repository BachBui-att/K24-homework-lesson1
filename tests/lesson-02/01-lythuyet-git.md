#Câu lệnh git init\
Tạo ra 3 vùng (Working Directory, Staging Area, Repository)\

#Câu lệnh git add <tên_file>\
Đưa file vào vùng Staging Area\

#Câu lệnh git add .\
Add những file còn lại trong vùng Working Directory vào Staging Area\

#Câu lệnh git commit -m"message"\
Đưa file từ vùng Staging Area vào Repository thành một phiên bản có tên "message"
